# Changelog

## [1.0.0] - 2025-12-01
### Added
- Телеграм-бот Dish Vision с анализом еды по фото (калорийность + рецепт).
- Бесплатный тариф с дневными лимитами.
- Премиум-тариф и разовая покупка дополнительных анализов через Telegram Stars.
- Система промокодов для активации/продления премиума.
- Учёт лимитов в БД (ежедневные и купленные анализы).
- Админ-функции: сброс лимитов, включение/выключение премиума, промокоды.

### Changed
- Тексты и бренд переименованы во «Dish Vision».
- Обновлено приветствие, помощь и профиль пользователя.

### Fixed
- Корректный учёт купленных анализов + бесплатных дневных лимитов.
