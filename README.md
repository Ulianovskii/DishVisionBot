# Food Lens — Telegram-бот для анализа еды по фото

Этот репозиторий содержит техническое задание (ТЗ), промпты и основные артефакты
для разработки Telegram-бота **Food Lens**.

## Содержимое

- `docs/01_overview.md` — общее описание продукта и тарифов
- `docs/02_navigation_and_fsm.md` — кнопки, экраны и FSM
- `docs/03_user_stories.md` — пользовательские истории (US1–US7)
- `docs/04_db_schema.md` — структура БД (черновой вариант)
- `docs/05_openai_prompts.md` — контракты с OpenAI (промпты)
- `docs/06_abuse_protection.md` — защита от злоупотреблений и лимитов
- `docs/07_nonfunctional_requirements.md` — нефункциональные требования

Реализация бота (код) предполагается в структуре:

- `app/handlers/` — обработчики Telegram-апдейтов
- `app/services/` — доменная логика (лимиты, отчеты, работа с GPT)
- `app/repositories/` — слой доступа к БД
- `app/locales/ru/` — тексты и кнопки для русского интерфейса
- `app/prompts/` — системные промпты для GPT

## Как использовать

1. Скопировать структуру `docs/` в проект.
2. На основе этого ТЗ реализовывать слои:
   - FSM и handlers в Aiogram
   - Сервис работы с GPT на базе `app/prompts/food_analysis.py`
   - Слой хранения и лимитов на PostgreSQL.
3. Обновлять документы в `docs/` по мере развития проекта.
